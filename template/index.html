Welcome to the Healthcare Prediction System

This intelligent healthcare assistant uses Machine Learning models to predict the likelihood of common diseases based on your health data. It currently supports prediction for Diabetes, Heart Disease, and Parkinson's Disease.

Early detection is key to prevention and treatment. This system helps patients and healthcare professionals take proactive steps towards better health by providing accurate predictions, visual insights, and educational information about each disease.

Diabetes is a chronic condition where the body cannot properly process sugar (glucose) due to insufficient insulin production or resistance. Left unchecked, it can damage vital organs and increase the risk of heart disease, kidney failure, and vision problems.

Common Symptoms:

  • Frequent urination and excessive thirst
  • Unexplained weight loss
  • Fatigue and weakness
  • Slow-healing wounds and frequent infections
  • Blurred vision

About the Project

This project is built using Flask for the web interface and Machine Learning models for prediction. Each disease model is trained on real-world datasets and evaluated using metrics such as accuracy, precision, recall, and F1 score.

The system is designed for educational purposes and as a tool for early awareness. It allows users to enter simple health parameters and receive predictions, along with visual insights like confusion matrices and feature importance charts.

By combining machine intelligence with clear, understandable information about symptoms and effects, this project encourages proactive healthcare and informed decisions.
